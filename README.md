## hw-report
---
### To setup:
```bash
    python -m venv <env_name>

    source <env_name>/bin/activate  // linux/macos
    <env_name>\Scripts\activate.bat // windows

    pip install -r requirements.txt
```

### Running app

```bash
    python -m app
```
